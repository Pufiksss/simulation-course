### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

---

## Реализация — линейный конгруэнтный генератор

Реализован **линейный конгруэнтный генератор** по формуле:

```
x_{n+1} = (a * x_n + c) mod m
alpha_n  = x_n / m
```

Параметры взяты из стандарта ANSI C:

| Параметр | Значение | Роль |
|---|---|---|
| `m` | 2³¹ − 1 = 2 147 483 647 | Модуль — максимальное состояние, простое число Мерсенна |
| `a` | 1 103 515 245 | Множитель — «перемешиватель» значений |
| `c` | 12 345 | Приращение — сдвиг, предотвращает зацикливание на 0 |
| `x` | seed = 42 | Начальное состояние генератора |

```python
class BasicRNG:
    def __init__(self, seed=42):
        self.m = 2**31 - 1
        self.a = 1103515245
        self.c = 12345
        self.x = seed

    def next(self):
        self.x = (self.a * self.x + self.c) % self.m
        return self.x / self.m

    def generate(self, n):
        return [self.next() for _ in range(n)]
```

---

## Теоретические значения

Базовый датчик генерирует числа, равномерно распределённые на [0, 1]

Для него существуют точные формулы:

**Математическое ожидание:**
```
M[X] = (a + b) / 2 = (0 + 1) / 2 = 0.5
```

**Дисперсия:**
```
D[X] = (b - a)² / 12 = 1² / 12 ≈ 0.08333
```

## Проверка требований к базовым датчикам

### 1. Равномерное распределение

Каждое значение в диапазоне [0, 1] равновероятно.

```python
my_mean = sample_mean(my_sample)     # 0.49934  (теория: 0.5)
my_var  = sample_variance(my_sample) # 0.08293  (теория: 0.08333)
```

### 2. Отсутствие корреляции

Каждое следующее число не зависит от предыдущего.:

```python
pairs_x = my_sample[:-1]
pairs_y = my_sample[1:]
mean_x  = sample_mean(pairs_x)
mean_y  = sample_mean(pairs_y)
cov     = sum((a - mean_x) * (b - mean_y)
              for a, b in zip(pairs_x, pairs_y)) / len(pairs_x)
corr    = cov / (sample_variance(pairs_x)**0.5 * sample_variance(pairs_y)**0.5)
# corr ≈ 0.0003  → корреляции нет
```

Автокорреляция ≈ 0 — требование выполнено.

### 3. Апериодичность

Генератор периодичен по природе, но период максимально длинный.

Параметры `a`, `c`, `m` удовлетворяют **теореме Халла-Добелла**, период = m = **2 147 483 647**.

Наша выборка (100 000 чисел) составляет 0.005% от периода — повторений нет.

### 4. Воспроизводимость

Одинаковый seed - одинаковая последовательность всегда:

```python
rng1 = BasicRNG(seed=42)
rng2 = BasicRNG(seed=42)
print(rng1.generate(3) == rng2.generate(3))  # True
```

### 5. Быстрота

Одна итерация — три арифметических операции (умножение, сложение, остаток).

```python
import time
start = time.time()
BasicRNG(seed=42).generate(100_000)
print(f"Время генерации 100 000 чисел: {time.time() - start:.4f} сек")
# ~0.05 сек
```

## Вывод

Реализованный линейный конгруэнтный генератор удовлетворяет всем требованиям к базовым датчикам случайных чисел:

- выборочные среднее и дисперсия совпадают с теоретическими значениями для U(0,1)
- автокорреляция между соседними значениями близка к нулю
- период генератора (~2 млрд) многократно превышает размер выборки
- при фиксированном seed результаты полностью воспроизводимы
- генерация 100 000 значений происходит в кратчайшие сроки благодаря простым операциям одной итерации

Небольшие отклонения от теоретических значений объясняются конечным размером выборки — следствие закона больших чисел. При увеличении размера выборки мы будем получать более точные результаты по отношению к теоритическим.
